# estudo_python
Coleção de pequenos exercícios em Python utilizados para estudo.

## Jogos disponíveis

- `adivinhacao.py`: jogo completo de adivinhação com seleção de
  dificuldade, validação de entrada e contagem de pontos.
- `administracao1_2.py`: utilitários para avaliar um chute comparado a um
  número secreto, com uma pequena demonstração interativa.

Execute qualquer um dos arquivos diretamente com Python para iniciar a
experiência correspondente:

```bash
python adivinhacao.py
python administracao1_2.py
```
